# Anta (Archived)

This project has been merged into [Capivara](https://github.com/xAngryBadger/capivara) as the **Compress PDF** tool.

Capivara is a full PDF suite with 15 tools (compress, merge, split, convert, OCR, and more). The compression functionality from Anta lives on there with improvements and a redesigned interface.

Live tool: [xangrybadger.github.io/capivara](https://xangrybadger.github.io/capivara/)

---

## What was Anta

PDF compressor with quality slider, real-time reduction estimate and result preview.

## Stack

- React 19 + Vite
- Python FastAPI
- pypdf
